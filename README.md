# ListMaster AI — مساعد تحضير قوائم التسجيل

منصّة لتكوين أقسام السنة الدراسية القادمة وتوزيع التلاميذ عليها — **معالجة محلّية بالكامل داخل المتصفح** (لا يُرفَع أيّ ملفّ إلى أيّ خادم).

> 🔒 **نموذج تجريبي محميّ بكلمة مرور.**

## المزايا

- استيراد قوائم التلاميذ من ملفّات Excel (‎.xls / .xlsx‎) عبر SheetJS.
- تكوين الأقسام وضبط معايير التوزيع.
- توزيع آليّ متوازن للتلاميذ على الأفواج.
- مراجعة وتعديل يدويّ للنتائج.
- تصدير القرار النهائي إلى Excel.

## التقنيات

موقع ثابت بلا أداة تجميع (no-build):

- React 18 + ReactDOM (UMD)
- Babel Standalone (تحويل JSX في المتصفح)
- Tailwind CSS (CDN) + نظام تصميم مخصّص (`design-system.css`)
- SheetJS (`xlsx`) لقراءة/كتابة Excel
- خطّ Cairo

## التشغيل محلّيًا

```bash
python -m http.server 8010
# ثم افتح: http://localhost:8010
```

## الخصوصيّة

لا يحتوي هذا المستودع على أيّ بيانات تلاميذ. جميع ملفّات Excel مُستبعَدة عبر `.gitignore`،
والمعالجة تتمّ بالكامل في متصفّح المستخدم دون إرسال أيّ بيانات إلى الخارج.
